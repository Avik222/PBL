
This script recommends movies and music based on your vibe.

## Installation

1.  Clone the repo.
2.  Install requirements:
    ```bash
    pip install -r requirements.txt
    ```

## Setup

You will need API keys from Google Gemini. Set them as environment variables:


-  `GEMINI_API_KEY`

## Usage

Run the script:
```bash
python your_script_name.py
